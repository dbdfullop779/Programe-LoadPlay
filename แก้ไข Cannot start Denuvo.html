คู่มือแก้ปัญหาเข้าเกม (ฉบับจับมือทำ)

ทำตามลำดับขั้นตอนที่ 1 ถึง 6 อย่างละเอียดดังนี้ครับ

1
ย้ายไฟล์เกม (สำคัญมาก)

เข้าไปที่โฟลเดอร์เกมที่คุณโหลดมา หาโฟลเดอร์ที่ชื่อ DenuvOwO
ให้ ก๊อปปี้ไฟล์ทั้งหมด ที่อยู่ในนั้น ออกมาวางไว้ที่ โฟลเดอร์หลักที่ติดตั้งเกม (โฟลเดอร์ที่มีไฟล์ .exe ของเกมนั่นเองครับ)

2
ตั้งค่าเปิดระบบใน BIOS

ขั้นตอนนี้ต้องทำที่ตัวเครื่องก่อนเข้า Windows:
  • สั่ง Restart คอมพิวเตอร์ แล้วกดปุ่ม Del หรือ F2 ย้ำๆ ขณะเครื่องกำลังเริ่มติด
  • หาเมนูชื่อ Intel Virtualization Technology (VT-x) หรือ SVM Mode
  • เปลี่ยนค่าให้เป็น Enabled
  • กดปุ่ม F10 เพื่อ Save และ Restart กลับเข้าหน้า Windows ปกติ

3
ปิดระบบ Hyper-V และ Core Isolation

เมื่อเข้าหน้า Windows แล้วให้ทำตามนี้:
  1. พิมพ์ในช่องค้นหา (Search) ว่า Turn Windows features on or off แล้วกด Enter
  2. หาบรรทัดที่เขียนว่า Hyper-V แล้วคลิกเอาเครื่องหมาย ติ๊กถูกออก ให้หมด แล้วกด OK
  3. ไปที่ Settings > Windows Security > Device Security > Core isolation details
  4. กดปิด Memory integrity ให้เป็น Off

4
ใช้คำสั่งปิดระบบ Hypervisor

  1. พิมพ์ค้นหาคำว่า CMD คลิกขวาเลือก Run as Administrator (เรียกใช้ในฐานะผู้ดูแล)
  2. พิมพ์หรือก๊อปปี้คำสั่งนี้ลงไปแล้วกด Enter:
    3. bcdedit /set hypervisorlaunchtype off

5
รันสคริปต์ VBS และเลือกตัวเลือก

ไปที่โฟลเดอร์เกมที่คุณย้ายไฟล์มาวางไว้แล้ว:
  1. คลิกขวาที่ไฟล์ VBS.cmd เลือก Run as Administrator
  2. เมื่อหน้าจอสีดำเด้งขึ้นมา ให้กดเลข 1 แล้วกด Enter (เพื่อเลือก Continue)
  3. จากนั้นกดเลข 1 อีกครั้ง แล้วกด Enter (เพื่อเลือก Restart Now)
  4. *จังหวะสำคัญ:* ขณะเครื่องกำลังเริ่มใหม่ (ก่อนจะเห็นโลโก้ Windows) ให้กดปุ่ม F7 บนคีย์บอร์ดย้ำๆ
  5. จะเห็นเมนูขึ้นมา ให้เลือก Disable Driver Signature Enforcement

6
วิธีรันเข้าเล่นเกม

เมื่อเข้าหน้า Windows ได้แล้ว ให้เข้าโฟลเดอร์เกมแล้วเปิดเกมผ่านไฟล์ใดไฟล์หนึ่งด้านล่างนี้เท่านั้น:
  • ไฟล์ชื่อ hypervisor-launcher.exe
  • หรือไฟล์ชื่อ steamclient_loader_x64.exe
⚠️ คำแนะนำสุดท้าย:
หลังจากเล่นเกมเสร็จแล้ว หากต้องการคืนค่าความปลอดภัยให้คอมพิวเตอร์ตามเดิม ให้รันไฟล์ VBS.cmd อีกครั้งแล้วเลือกคำสั่ง Revert Changes ครับ
