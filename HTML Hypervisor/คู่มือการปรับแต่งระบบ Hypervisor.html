🛠️ คู่มือการปรับแต่งระบบ Hypervisor

โปรดทำตามขั้นตอนให้ครบถ้วน เพื่อการใช้งานที่สมบูรณ์

1 เปิดระบบ Virtualization ใน BIOS

ขั้นตอนนี้ต้องทำ "ก่อน" เข้าหน้า Windows เพื่อเปิดการทำงานจากระดับเครื่อง:

  • สั่ง Restart คอมพิวเตอร์ของคุณ
  • ขณะเครื่องกำลังเริ่มติด (จังหวะจอดำ) ให้กดปุ่ม Del หรือ F2 ย้ำๆ จนเข้าหน้าเมนู BIOS
  • ไปที่เมนู Advanced หรือ CPU Configuration แล้วมองหาหัวข้อ:
    • ถ้าใช้ Intel: Intel Virtualization Technology หรือ VT-x
    • ถ้าใช้ AMD: SVM Mode
  • เปลี่ยนค่าให้เป็น Enabled
  • กดปุ่ม F10 เพื่อ Save & Exit เครื่องจะกลับเข้า Windows ปกติ

2 ปิดระบบ Hyper-V และ Core Isolation

เมื่อกลับมาที่หน้า Windows แล้ว ให้ปิดระบบรักษาความปลอดภัยบางตัวที่ขัดขวางการทำงาน:

  1. คลิกที่ช่องค้นหา (Search) พิมพ์ว่า Turn Windows features on or off แล้วกด Enter
  2. มองหาบรรทัดชื่อ Hyper-V ให้ คลิกเอาเครื่องหมายถูกออก ให้หมด แล้วกด OK
  3. ไปที่ Settings > Windows Security > Device Security
  4. คลิกที่หัวข้อ Core isolation details แล้วกด ปิด (Off) ตรงเมนู Memory integrity

3 ใช้คำสั่งปิด Hypervisor Launch

ขั้นตอนสุดท้าย คือการสั่งปิดระบบจำลองผ่านหน้าจอคำสั่ง (CMD):

  1. พิมพ์ค้นหาคำว่า CMD ในช่องค้นหา
  2. คลิกขวา ที่ไอคอน Command Prompt แล้วเลือก Run as Administrator (เรียกใช้ในฐานะผู้ดูแล)
  3. คัดลอกคำสั่งด้านล่างนี้ไปวางในหน้าจอสีดำ แล้วกด Enter:
    4. bcdedit /set hypervisorlaunchtype off
  4. เมื่อขึ้นคำว่า The operation completed successfully ให้ทำการ Restart เครื่องอีกครั้งหนึ่ง เป็นอันเสร็จสิ้นครับ

⚠️ ข้อควรรู้และแนวทางปฏิบัติที่ปลอดภัย (สำคัญมาก)

  • คืนค่าความปลอดภัยทุกครั้ง: เมื่อเล่นเกมเสร็จแล้ว ควรเรียกใช้ไฟล์ VBS.cmd และเลือก Revert Changes พร้อม Restart เพื่อเปิดระบบป้องกันของ Windows กลับมาตามเดิมทันที
